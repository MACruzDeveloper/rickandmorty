# UI-FRONT

> Frontend test Rick and Morty api

## Env Setup
  - node: `20.10.0`
  - npm: `10.2.3`

## Commands
  - `npm install` to install all needed packages 
  - `npm run dev` - Will start the dev mode and open a server `localhost:5173`

## Code pattern
  - React
  - Redux (toolkit)
  - Vite
  - Node
  - Tailwind css
